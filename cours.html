Cours git


Initialisation

la commande pour initialiser git sur un dossier par exemple est 'git int'

Configuration

Git config

La commande pour configiurer git, c'est à dire configurer soit globalement soit sur un projet par exemple en global est git config --global user.email ou encore git config --global user.name

manipulation des fichers

Git add

La commande pour ajouter un ou des fichiers au repertoire initialisé avec git, c'est à dire ajouter soit tous les fichiers du projet soit un seule est
git add . ou git add "nom fichier"
